
## This project is using aspnet mvc as backend & react, webpack v3 as frontend

**Commands to run/start project**
> 
  * npm install
  * npm start  (localhost:3000)
  * npm run build (publish to dist folder)

**Frontend stack**
> 
  * webpack 3
  * react
  * redux
  * sass
  * sagas
  * redux-action

**Backend stack**
> 
  * aspnet core 1.1
  * vs 2017
